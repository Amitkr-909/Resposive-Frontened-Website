# 🎓 Responsive Education Website

A modern and fully responsive education website built using HTML, CSS, and JavaScript.
This project showcases a clean UI design suitable for schools, colleges, or online learning platforms.

---

## 🚀 Features
- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🧭 Navigation bar with smooth scrolling
- 🏫 Sections like Services, About, Contact, etc.
- 🎨 Clean and modern UI
- ✨ Interactive elements using JavaScript
- 📩 Contact form UI

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript
